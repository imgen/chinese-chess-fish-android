## 介绍

为什么叫象棋鱼呢？因为它使用了皮卡鱼引擎，同时它的目标是提供一个开源免费的象棋学习工具！

#### 问题：我该下载哪个apk？

有两个版本：

1. chessfish-armv8-dotprod.zip
2. chessfish-armv8.zip

如果你的手机是最近几年买的，你的CPU应该支持dotprod技术，那么就下载chessfish-armv8-dotprod.zip。

dotprod没法正常工作，就下载chessfish-armv8.zip。
