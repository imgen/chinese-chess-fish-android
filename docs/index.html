象棋鱼

开源免费的安卓版象棋学习工具

为什么叫象棋鱼呢？因为它使用了皮卡鱼引擎，同时它的目标是提供一个开源免费的象棋学习工具！

本工具有如下特色功能：

  • 开源免费;
  • 使用了皮卡鱼象棋引擎，棋力强大;
  • 显示历史轨迹(图1)
  • 选择棋子后，显示可移动位置(图2)
  • 己方出着时可寻求引擎帮助，可视化推荐着法，一键选择着法(图3)
  • 强制电脑变着，可视化显示变着，一键选择变着(图4)
